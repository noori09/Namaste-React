Q1. What is Prop- Drilling?
Ans- When we pass the data from parent to child and then to its child and so on. This process is known as
     prop-drillling.

Q2. What is lifting the state up?
Ans- When we give the control of child components to its parent component. The process is known as lifting the
     state up.

Q3. What is Context Provider & Context Consumer?
Ans- Context Provider is basically used to override the default value of the context. Context Consumer is used
     in class based components , if we want to use the value of context inside it.

Q4. If you don't pass the value to a provider does it take the default value?
Ans- No, it will throw an error, passing a value is mandatory in the Provider.
